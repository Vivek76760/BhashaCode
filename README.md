# 🇮🇳 BhashaCode (भाषाकोड)

BhashaCode is an open-source, multi-lingual system programming language engine designed to eliminate the English barrier in computer science education across India. 

Instead of forcing students to learn English syntax before logical programming, BhashaCode allows developers to write native execution logic in **Hindi, Tamil, Telugu**, and more, translating regional scripts into a unified compiler path.

## 🚀 Current Milestone (v3.0)
Our prototype engine is fully functional and natively supports:
- **Multi-lingual Tokenizer** Map (Regional words match universal execution tokens)
- **Dynamic Variable Management** (`मान`, `மாறி`, `రఖో`)
- **Expression Evaluation & Math Parsing** (`क = ख + १०`)
- **Logical Execution Loops** (`जबतक ... करो ... बन्द`)

## 🤝 How You Can Contribute For Free
We want to expand this to support all 22 official languages of India. You can contribute right now by:
1. **Adding Your Mother Tongue:** Open the dictionary map inside our script and add your language's native words for keywords like `LET`, `IF`, `WHILE`, and `PRINT`.
2. **Improving the Parser:** Helping us optimize syntax handling and block structures.
3. **Ecosystem & Tools:** Creating simple syntax-highlighting plugins for code editors.
